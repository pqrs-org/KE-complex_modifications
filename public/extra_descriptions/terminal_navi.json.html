Keyboard shortcuts for Terminal apps

Makes standard macOS keyboard shortcuts work in terminal apps.

Included key bindings

Input Ouput Notes
⌘Z Undo
⌘⇧Z Redo Zsh is the only shell that supports redo, but unfortunately, it's not bound in the default keymap. To enable this shortcut, you need to add the following to your ~/.zshrc file: 
bindkey "^[^R" redo
⌘D Duplicate last word Not bound by default in Bash/Readline.
⌥← Move cursor one word backward
⌥→ Move cursor one word forward
⌘← Move cursor to beginning of line
⌘→ Move cursor to end of line
⌥⌫ Delete one word backward
⌥⌦ Delete one word forward
⌘⌫ Delete to beginning of line

In Zsh, this shortcut will delete the whole line by default. To get it to work correctly, you need to add the following to your ~/.zshrc file:.

bindkey "^U" backward-kill-line
⌘⌦ Delete to end of line

Additional notes

This mod supports English and Finnish keyboards. If you want to use a different keyboard layout, please download the source code and modify terminal_navi.json.erb.
