Keyboard shortcuts for Terminal apps

Makes the following keyboard shortcuts work the same or similarly in both macOS and terminal apps:

Shortcut Action Bash/Readline Zsh
⌘Z Undo ✅ ✅
⇧⌘Z Redo ❌ requires bindkey '^[e' redo
⌘D Duplicate the word left of the cursor ❌ ✅
⇧⌘D Duplicate the last word of the previous command line ✅ ✅
⌫ Delete one character backward ✅ ✅
⌦ or fn⌫ Delete one character forward ✅ ✅
⌥← Go one natural word backward ✅ requires WORDCHARS=''
⌥→ Go one natural word forward ✅ requires WORDCHARS=''
⌥⌫ Cut one natural word backward ✅ requires WORDCHARS=''
⌥⌦ or ⌥fn⌫ Cut one natural word forward ✅ requires WORDCHARS=''
⌘← Go one shell word backward ✅ requires bindkey '^[^B' vi-backward-blank-word
⌘→ Go one shell word forward ✅ requires bindkey '^[^F' vi-forward-blank-word
⌘⌫ Cut one shell word backward ✅ requires zsh-edit
⌘⌦ or ⌘fn⌫ Cut one shell word forward ✅ requires zsh-edit
⌃A Go to previous line break ✅ ✅
⌃E Go to next line break ✅ ✅
⌃U Cut to previous line break ✅ requires bindkey '^U' backward-kill-line
⌃K Cut to next line break ✅ ✅
⌥↑ Go to first line ✅ first line of history ✅ first line of buffer or, if already there, of history
⌥↓ Go to last line ✅ last line of history ✅ last line of buffer or, if already there, of history
