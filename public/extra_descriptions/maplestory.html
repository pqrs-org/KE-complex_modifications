When MapleStory focused, use F1, F2, etc. keys as standard function keys

While the MapleStory application is focused, press the fn key to use the special features on each key. The standard function keys will also work while ⇪ caps lock, ⌘ command, ⌃ control, ⌥ option, or ⇧ shift are held down.
