Navigate with your left hand fingers:

Inspired by the "Wordstar Diamond", this modification lets you navigate with the Caps Lock key and the keys right next to it.

The Caps Lock key is remapped to the Function (Fn) key. You can then use the following keys:

  • Caps Lock + e: Up arrow
  • Caps Lock + s: Left arrow
  • Caps Lock + d: Right arrow
  • Caps Lock + x: Down arrow
  • Caps Lock + a: Jump to start of word
  • Caps Lock + f: Jump to end of word
  • Caps Lock + r: Page up
  • Caps Lock + c: Page down
  • Caps Lock + q: Jump to start of line
  • Caps Lock + w: Jump to end of line

You can use these combinations and press Shift to select text.

Read more about the "Wordstar Diamond" on this blog post by Ben Hoyt who inspired me to create this modification.
