［ US ］左右のコマンドキー（⌘）を、単独で押したときは 英数・かな キー として扱う（左⌘は 英数、右⌘は かな）

左右のコマンドキー（⌘）が、単独で押したときだけ 英数・かな キー として機能するようになります。

例:

  • 左⌘を単独で押したとき → 英数
  • 左⌘を押しながら C → ⌘C

  • 右⌘を単独で押したとき → かな
  • 右⌘を押しながら V → ⌘V

これにより、JIS 配列と同様にステートレスな 英数・かな 切り替えを実現できます。

注意:

  1. ⌘キーを単独で押した後に W を素早く押しすぎると、⌘W として認識されてしまいます。
    頻発するようであれば、同梱の「⌘Wを2連打したときのみタブまたはウィンドウが閉じるようにする」の設定も有効にしてください。
  2. ⌘キーを単独で押した後に Q を素早く押しすぎると、⌘Q として認識されてしまいます。
    頻発するようであれば、同梱の「⌘Qを2連打したときのみアプリが閉じるようにする」の設定も有効にしてください。
［ JIS ］英数・かな キーを、他のキーと組み合わせて押したときは コマンドキー（⌘）として扱う（単独では 英数・かな）

英数・かな キーが、他のキーと組み合わせて押したときだけ コマンドキー（⌘）として機能するようになります。

例:

  • ｢英数」を単独で押したとき → 英数
  • ｢英数」を押しながら C → ⌘C

  • ｢かな」を単独で押したとき → かな
  • ｢かな」を押しながら V → ⌘V

これにより、⌘キーを、よりホームポジションに近い位置で使えるようになります。
また、もともとの⌘キーが必要なくなるため、Simple Modifications を使って、その位置に好きな機能を割り当てることもできます。
