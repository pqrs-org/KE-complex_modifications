How it works

By default on macOS, function keys (F1-F12) act as media/system keys (brightness, volume, etc.). This rule overrides that so F keys always behave as true function keys inside supported applications, with no need to hold Fn.

The behaviour depends on two things: whether the active app is in the supported list, and whether you are using a Logitech keyboard.

Behaviour by scenario

Scenario Keys Result
Supported app - any keyboard F1-F12 True function key. Karabiner passes the keypress through unchanged.
Non-supported app - any keyboard F1, F2 Media key. F1 = brightness down, F2 = brightness up.
Non-supported app - non-Logitech keyboard F3-F12 Media key (see mapping table below). Karabiner remaps these to system actions.
Non-supported app - Logitech keyboard F3-F12 Passed through unchanged. Logi Options+ handles the media mapping natively, so Karabiner leaves these alone to avoid conflicts.
