Use Function Keys in supported applications, else use Media Keys

These keybindings ensure that F keys work as intended in supported applications.

In all other applications, the F keys will work as media keys.

Current list of supported applications where function keys will work as intended:

  • VS Code
  • Jetbrains Goland
  • Jetbrains Rider
  • Jetbrains PyCharm
  • Jetbrains IntelliJ IDEA
  • Jetbrains PhpStorm
  • Jetbrains CLion
  • Jetbrains WebStorm
  • Jetbrains RubyMine
  • Jetbrains Datagrip
  • Jetbrains RustRover
  • Eclipse
  • XCode
  • iTerm2
  • Terminal
  • Adobe Photoshop
  • Adobe Premiere Pro
  • Adobe Illustrator
  • Adobe Audition
  • Adobe After Effects
  • Davinci Resolve
  • Textmate
