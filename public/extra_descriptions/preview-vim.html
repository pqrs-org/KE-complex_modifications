vim-preview

This modification provides zathura/vim-like functionality to Preview.app.

Key binds currently implemented:

Movement

  • Left => h
  • Right => l
  • Down => j
  • Up => k
  • Previous position => H
  • Next position => L
  • Previous page => K
  • Next page => J
  • First page => g
  • Last page => G

Zoom

  • Zoom In => =
  • Zoom Out => -
  • Actual size => 0
  • Zoom to fit => 9
The main repository is hosted on Github. Contributions are welcome!
