Use Function Keys in Developer Apps, else use Media Keys

These keybindings ensure that F keys work as intended in developer applications.

In all other applications, the F keys will work as media keys.

Current list of developer applications:

  • VS Code
  • Jetbrains Goland
  • Jetbrains Rider
  • Jetbrains PyCharm
  • Jetbrains IntelliJ IDEA
  • Jetbrains PhpStorm
  • Jetbrains CLion
