Dual keys

Relieve stress from you pinkies by moving Escape, Enter, -, =, ', `, [ and ] to the modifier keys, without losing the ability to use them as modifier keys.

Notes

  • Using the Shift keys as a dual keys is mainly intended for keyboards that let you access them with your thumbs. You can use them on regular keyboard, too, but you won't to get as much benefit from it.
  • Likewise, if your Control keys are located on the outer edges of your keyboard (because it has a wide spacebar), then it's not recommendeded to use them as dual keys.
  • This mod is made for US Qwerty keyboards. If you want to use a different keyboard layout, or otherwise remap the keys, please download the source code and modify dual_keys.json.rb.
