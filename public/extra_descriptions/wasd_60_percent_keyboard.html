Recreate the WASD 60% keyboard experience on your system

Uses the Caps Lock key in combination with your right hand to recreate the control keys.

  • Caps Lock + i: Up arrow
  • Caps Lock + j: Left arrow
  • Caps Lock + l: Right arrow
  • Caps Lock + k: Down arrow
  • Caps Lock + u: Page up
  • Caps Lock + o: Page down
  • Caps Lock + h: CMD + Right Arrow (Home)
  • Caps Lock + n: CMD + Left Arrow (End)

These shortcuts work with combinations of ctrl, option, command, etc.

NOTE: Home and End use the CMD button and direction key to better reflect Mac Keyboards and commands. An example of the difference can be seen when entering text into web pages.
