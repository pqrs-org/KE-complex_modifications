Requirements

This rule depends on the macOS Input Sources shortcuts. Before enabling the rule, open:

System Settings > Keyboard > Keyboard Shortcuts > Input Sources

Select the previous input source Control + Space
Select next source in Input menu Control + Shift + /

Input source order

From an English input source, Control + Space sends the macOS "next source in Input menu" shortcut, so it switches to the next input source in your macOS input source order.

From a non-English input source, Control + Space selects English directly. If your English input source is not detected, use Karabiner-EventViewer to inspect it and replace the English matcher in the JSON with a more specific input_source_id.
