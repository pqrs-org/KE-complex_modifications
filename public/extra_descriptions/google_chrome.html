Hyper Key

Includes the Caps Lock hyper key as the foundation for all Chrome shortcuts in this pack. Caps Lock is remapped to a unique modifier combination (⌃⇧⌘) that doesn't conflict with any built-in Chrome or macOS shortcut, giving you a dedicated left-hand anchor key that's fast to reach without lifting your hand off the home row.

All shortcuts in this pack are built on top of the hyper key, so activating them is a single fluid motion - hold Caps Lock and tap a letter - rather than awkward multi-finger contortions across the keyboard.

Caps Lock Maps to left control + left shift + right command (⌃⇧⌘). Used as the hyper key for all Chrome shortcuts in this pack. The Caps Lock function is not lost - use double-tap right shift instead.
Double-tap right shift Toggle caps lock on/off

Tab Management

Shortcuts for managing tabs without reaching for the mouse. Switch between recent tabs, duplicate the current tab, or pop a tab out into its own window - all from the home row.

Caps Lock Hyper Key + a Search tabs - quickly find and switch to any open tab by name
Caps Lock Hyper Key + w Toggle between last used tabs - jump back and forth between your two most recent tabs (requires CLUT: Cycle Last Used Tabs extension)
Caps Lock Hyper Key + e Duplicate the current tab - useful for keeping your original page open while exploring a new path (requires Duplicate Tab Shortcut extension)
Caps Lock Hyper Key + t Move the current tab to a new window - ideal for side-by-side comparisons or separating work contexts (requires Pop out tabs to new window extension)

Browser Actions

Common browser actions mapped to the hyper key for fast, consistent access without hunting through menus.

Caps Lock Hyper Key + q Switch between Google Chrome profiles
Caps Lock Hyper Key + r Reload the current page
Caps Lock Hyper Key + s Select the address bar - jump straight to the URL for quick navigation or copying
Caps Lock Hyper Key + d Clear browsing history
Caps Lock Hyper Key + f Search the web - focuses the address bar ready for a new search query
Caps Lock Hyper Key + g View downloads

Developer Tools

One-key access to Chrome's developer tools panels, so you can jump straight to the panel you need instead of navigating through menus or remembering separate shortcuts for each panel.

Caps Lock Hyper Key + 1 Open developer tools
Caps Lock Hyper Key + 2 Open developer console
Caps Lock Hyper Key + 3 Open developer elements panel - inspect and modify the DOM and CSS
Caps Lock Hyper Key + 4 View page source
Caps Lock Hyper Key + q Toggle device toolbar (when a developer panel is already open) - simulate different screen sizes and devices
