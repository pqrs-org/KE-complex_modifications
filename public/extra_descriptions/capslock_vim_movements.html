Caps Lock Vim Movements

Simulate Vim movements with caps lock as the main modifier.

Motivation

After trying a few different rules, they were too complicated and the feature set ended up not combining well with native MacOS modifiers.

The objectives of this rule set are:

  • Home-row caps lock engagement.
  • No modifier overrides so OS defaults continue to work.
  • Do not map keys that OS (MacOS) already support; i.e. Home can be achieved with Command + Caps + K.
Key Map
Key Maps To
Caps lock Escape
Caps lock + Return Caps lock
Caps lock + H Left arrow
Caps lock + J Down arrow
Caps lock + K Up arrow
Caps lock + L Right arrow
Caps lock + D Page Down
Caps lock + U Page Up
