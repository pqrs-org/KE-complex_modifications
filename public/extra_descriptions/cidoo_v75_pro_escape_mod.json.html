CIDOO V75 Pro — Escape & Tilde Swap Mod w/ Hyper Key

This rule is for the CIDOO V75 Pro and designed to mimic the Keychron K6.

⚠️ This rule assumes the Escape (Esc) and Tilde (~) keys have been physically swapped and reconfigured as such in VIA.

It also adds the Hyper Key mod (⌃⌥⇧⌘) to Caps Lock when held (tap for regular Caps Lock).

This is built upon my existing Keychron K6 rules.

This matches the key positioning for users who also use the built-in MacBook keyboard.

Full list of modifications:

  • Unmodified Esc: Escape
  • Left Shift + Esc: ~ (tilde)
  • Right Shift + Esc: ` (grave)
  • Control + Esc: Control + `
  • Command + Esc: ⌘ + ` (app window cycle)
  • Caps Lock (held): ⌃⌥⇧⌘ (Hyper Key)
  • Caps Lock (alone): Regular Caps Lock
  • Hyper Key + Esc: Shift + Escape
