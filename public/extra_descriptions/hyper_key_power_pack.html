Hyper Key Power Pack

Includes 3 hyper keys for maximum flexibility.

Caps Lock Maps to left control + left shift + right command (⌃⇧⌘). The Caps lock key functionality is disabled.
Right Option Maps to left shift + left option + left command (⌃⇧⌥⌘)
Tab (long press) Maps to left control + left command (⌃⌘). To use the tab key as intended, short press the tab key.

Toggling Caps lock

Double-tap right shift Caps lock toggle
