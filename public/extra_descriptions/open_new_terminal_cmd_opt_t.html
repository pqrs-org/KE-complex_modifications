Open a new Terminal window instantly

Press ⌘ + ⌥ + T to run a tiny AppleScript that asks Terminal to open a brand-new window using your default shell profile.

  • Uses the left Command + left Option modifiers to avoid conflicts with right-side shortcuts.
  • Works whether or not Terminal is already running and always requests a fresh window instead of just focusing the last one.
  • Creates a blank shell session so you can start typing immediately.
