Microsoft Sculpt Ergonomic Desktop (ISO PT-pt keyboard layout)

This set of complex modifications provides macOS-friendly customizations to Microsoft Sculpt Ergonomic Desktop (ISO PT-pt keyboard layout).

Customizations

  • Swap midifier keys, so that their arrangement matches that of Apple keyboards:
    • key to key
    • key to key
    • key to key
    • key to key
  • Map key to + .
  • Map key to + .
  • Map key to screencapture -i -W -U -c command.
  • Map key to + + (lock screen).
  • Map key to key.
  • Maps the top row of keys as controls for similar, built-in macOS features:
    • Map (F2) key to launching Mission Control.
    • Map (F3) key to Move left a space action.
    • Map (F4) key to Move right a space action.
    • Map (F5) key to launching Spotlight.
    • Map (F6) key to launching Calculator.
    • Map (F7) key to launching Screen Sharing.
    • Map (F8) key to launching System Preferences.
  • Maps the mouse button to + (previous page).
  • Maps the mouse button to launching applications list (the Launchpad).
  • Toggle and map key actions on Numpad. If NumLock enabled map Numpad keys actions: 7 = Home, 1 = End, 9 = Page Up, 3 = Page Down, 8 = Up arrow, 2 = Down arrow, 4 = Left arrow, 6 = Right arrow, 0 = Insert, . = Delete.
  • Maps the keys to the correct ISO PT-pt keyboard layout:
    • Fix (backslash) and (pipe) keys mapping.
    • Fix (less than) and (greater than) keys mapping.
    • Fix (tilde) and (circumflex) keys mapping.
    • Fix (male ordinal indicator) and (female ordinal indicator) keys mapping.
    • Fix (plus), (times) and (umlaut) keys mapping.
    • Fix and (guillemets) keys mapping.
    • Fix (right curly bracket) and (left curly bracket) keys mapping.
    • Fix (pound) key mapping.
    • Fix (section sign) key mapping.
    • Fix (euro) key mapping.

Usage

Toggle Fn Switch on Keyboard

Move the toggle on Microsoft Sculpt Ergonomic Keyboard to the blue position so that the top row of keys work as controls for built-in macOS features.

Sync CapsLock between build in and MS keyboard

Go to "Karabiner-Elements / Devices" and enable "Modify events" for Your MS keyboard.

Change Top Row of Keys to Work as Standard Function Keys

This puts Karabiner-Elements (as opposed to macOS) in charge of mapping key codes to controls for built-in macOS features. This also disables the default 'Function keys' mappings shipped with Karabiner-Elements so that customizations would not be overriden down the Input Event Modification Chain.

  • Choose Apple menu  > System Preferences.
  • Click Keyboard.
  • Click Keyboard Shortcuts.
  • Select Function Keys on the left side.
  • Enable Use F1, F2, etc. keys as standard function keys on the right side.
Disable display brightness keyboard shortcuts

If the Decrease display brightness or Increase display brightness keyboard shortcuts are enbaled and set to F14 and F15, respectively, it might stop ScrLk and Pause keys from working.

  • Choose Apple menu  > System Preferences.
  • Click Keyboard.
  • Click Keyboard Shortcuts.
  • Select Display on the left side.
  • Disable Decrease display brightness and Increase display brightness options on the right side.

Notes

  • The (F1) key and the key are impossible to map. This is due to macOS not being able to receive the scancodes from these keys.
  • Media playback control keys do not need special treatments as they work out of the box.
