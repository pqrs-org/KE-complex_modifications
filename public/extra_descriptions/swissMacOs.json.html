About

This configuration is a lightweight Swiss keyboard remap focused on developers. It moves frequently needed programming characters onto keys that are rarely used on the Swiss layout, making brackets and braces faster to reach without changing your overall layout.

BASIC MODE

Key remaps included:

  • Shift + ' → Right Option + 8
  • Shift + # (non-us pound) → Right Option + 9
  • Shift + \ → Right Option + 9
  • Shift + [ → Right Option + 5
  • Shift + ] → Right Option + 6
  • ] → Shift + ]
