Neo2 layout

Rules for the Neo2 keyboard layout.

These modifications provide better shortcut support. However, due to using the ABC Extended layout, not all keys are implemented. Layer 5 and 6 are also missing, so in order to use those the other Neo2 modifications need to also enabled.

Both modification sets can be enabled at the same time, by changing the layout between ABC Extended and the corresponding Neo layout.

To use a PC style keyboard, the following simple modifications need to be enabled:

grave_accent_and_tilde (`) -> non_us_backslash

non_us_backslash           -> grave_accent_and_tilde (`)

non_us_pound               -> backslash (\)

right_option               -> right_command
