Hyper Key

Includes the Caps Lock hyper key as the foundation for all Slack shortcuts in this pack. Caps Lock is remapped to a unique modifier combination (⌃⇧⌘) that doesn't conflict with any built-in Slack or macOS shortcut, giving you a dedicated left-hand anchor key that's fast to reach without lifting your hand off the home row.

All shortcuts in this pack are built on top of the hyper key, so activating them is a single fluid motion - hold Caps Lock and tap a letter or number - rather than awkward multi-finger contortions across the keyboard.

Caps Lock Maps to left control + left shift + right command (⌃⇧⌘). Used as the hyper key for all Slack shortcuts in this pack. The Caps Lock function is not lost - use double-tap right shift instead.
Double-tap right shift Toggle caps lock on/off

Navigation

Jump directly to the views you use most in Slack without clicking through the sidebar. Get to unread messages, DMs, and activity notifications instantly.

Caps Lock Hyper Key + a Quick search - fuzzy search across channels, people, and messages
Caps Lock Hyper Key + 1 Activity - view all recent notifications and mentions in one place
Caps Lock Hyper Key + 2 Direct messages - jump to your DM list
Caps Lock Hyper Key + 3 All unread messages - see every unread message across all channels at a glance

Message Formatting

Format messages without reaching for the toolbar. Useful when sharing code snippets, commands, or quoting someone's message in a reply.

Caps Lock Hyper Key + s Inline code format - wrap selected text in backticks for inline code
Caps Lock Hyper Key + d Code block - wrap selected text in a multi-line code block
Caps Lock Hyper Key + q Quote - format selected text as a block quote
Caps Lock Hyper Key + g Hyperlink - add a hyperlink to selected text

Actions

Common Slack actions mapped to the hyper key for quick access without hunting through menus.

Caps Lock Hyper Key + w Toggle sidebar - hide or show the channel sidebar to reclaim screen space
Caps Lock Hyper Key + e Set status - quickly update your Slack status
Caps Lock Hyper Key + f Upload file - open the file picker to attach a file to your message
