English

When you use an external ABNT2 (Brazilian Portuguese) keyboard on a Mac, the physical pipe / backslash key is reported as non_us_backslash and the grave / tilde key is reported as grave_accent_and_tilde. These two keys end up swapped compared to what is printed on the keycaps.

This rule swaps the two keys back so they type the expected characters. It works with any modifier held down (optional: ["any"]).

Physical key (from) Sends (to)
\ / | (non_us_backslash) ` / ~ (grave_accent_and_tilde)
` / ~ (grave_accent_and_tilde) \ / | (non_us_backslash)

Português

Ao usar um teclado externo ABNT2 (português do Brasil) em um Mac, a tecla física de barra / pipe é reportada como non_us_backslash e a tecla de crase / til é reportada como grave_accent_and_tilde. Essas duas teclas acabam trocadas em relação ao que está impresso no teclado.

Esta regra troca as duas teclas de volta para que digitem os caracteres esperados. Funciona com qualquer modificador pressionado (optional: ["any"]).

Tecla física (de) Envia (para)
\ / | (non_us_backslash) ` / ~ (grave_accent_and_tilde)
` / ~ (grave_accent_and_tilde) \ / | (non_us_backslash)
