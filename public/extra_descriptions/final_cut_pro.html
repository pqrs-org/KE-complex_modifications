A set of mouse and keyboard shortcuts for Final Cut Pro that make common editing tasks faster - scrolling the timeline, trimming clips, making cuts, and adjusting zoom - without having to switch tools manually.

Scroll the timeline with the mouse

Button 3 (middle-click, hold and drag) Switches to the hand tool and scrolls the timeline horizontally while held. Releasing the button automatically reverts back to the select tool.

Trim clips with the mouse

Left control + Button 1 (left-click) Trims the start of the clip to the playhead position (control + [)
Left control + Button 2 (right-click) Trims the end of the clip to the playhead position (control + ])

Cut clips with the mouse

Left cmd + Button 1 (left-click) Makes a cut at the click position. This automatically switches to the blade tool, performs the cut, then reverts back to the select tool.

Zoom shortcuts

Left option + Z Zooms the timeline to fit all clips in view (shift + z), then zooms out one level (cmd + -) for a bit of breathing room around the clips.
