Description

Right_command to korean <-> english

Notes

mac-os의 기본 한글 전환 방식은 key-up 이벤트(키를 누르고 땔때) 한글 전환이 된다
반대로, windows-os의 한글 전환 방식은 key-down(키를 누르자마자) 한글 전환이 된다
그래서 이 확장은 key-down 이벤트로 한영 전환을 할 수 있도록 구현하였다.
문제가 있다면, 저에게 email 주시면 감사하겠습니다.
추가적으로, 구름 입력기 두벌식 한글자판과 기본 입력기 영어 자판과 변환하는 기능도 제공한다.

Originally, the mac-os input-source-change event was triggered by a key-up event.
In contrast, the windows-os input-source-change event was triggered by a key-down event.
So, this function is windows-os liked input source change system.
