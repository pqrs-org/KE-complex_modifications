simvi - simple vi mode

from tekezo's vi, dosen't work with mac os's combination move keys

so I added keysets like below

for work with mac os's combination keys

  • option + s + h,j,k,l
  • shift + option + s + h,j,k,l
  • command + s + h,j,k,l
  • shift + command + s + h,j,k,l

highly recommended to vi + mac combination keys lover users
