Karabiner Auto Quotes

A complex modification that automatically closes brackets and quotes while typing. It does the following:

  • Typing a ( is automatically followed by a ) and placing the cursor between them
  • '
    • is followed by a '
  • "
    • by another "
  • `
    • by `
  • [
    • by ]
  • {
    • by }
  • “
    • by ”
  • ‘
    • by ’
  • <
    • by >

For normal behavior (i.e. to type a single one of these) hold fn. For example fn+' types one single quote.

It contains exceptions for Atom, iTerm, PyCharm and VSCodium, because all of these already have their own intelligent quote and bracket handling.
