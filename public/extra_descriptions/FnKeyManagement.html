Description

Fn Key reverse - for specific apps.

Notes

intellij 와 같이 mac-os에서 fn key를 기본값으로 단축키를 많이 사용하는 경우 불편하여 그러한 앱에서만 fn key 작동 방식이 반대로 동작하도록 구성하였다
예를들어 shift+f6 (이름바꾸기) 단축키를 사용하고자 할 때에 shift + fn + f6 이 아니라 intellij 에 한해서만 shift + f6 으로 동작한다.
실제 f6의 기능을 하고 싶으면, fn + f6 버튼을 누르면 된다
문제가 있다면, 저에게 email 주시면 감사하겠습니다.
