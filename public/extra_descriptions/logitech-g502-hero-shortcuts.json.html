Logitech G502 Hero Mouse Shortcuts

This rule adds shortcuts to Logitech G502 Hero Mouse to replace Logitech GHub's common controls

  • Button3 (Middle Mouse Button) → Mission Control
  • G4 → ← (Left Arrow)
  • G5 → → (Right Arrow)
G4 and G5 also support any modifiers so following options are possible:
  • G4 + Control → Previous Workspace
  • G5 + Control → Next Workspace
