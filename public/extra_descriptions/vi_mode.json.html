Moving Around

Trigger Key + H/J/K/L Arrow Keys
Trigger Key + F Fn
Trigger Key + F + H/J/K/L Home / Page Down / Page Up / End
Trigger Key + 0/4 Go to Start/End of the Line
Trigger Key + Command + H/L Go to Start/End of the Line
Trigger Key + Command + J/K Go to Start/End of the Document
Trigger Key + Option + H/L Go to Start/End of the Word

Selecting Text

Trigger Key + Shift + H/J/K/L Select Text
Trigger Key + Shift + Command + H/L Select Until Start/End of the Line
Trigger Key + Shift + Command + J/K Select Until Start/End of the Document
Trigger Key + Shfit + Option + H/L Select Until Start/End of the Word
Trigger Key + Shfit + Option + J/K Select Until Start/End of the Paragraph

Visual Mode

V + H/J/K/L Select Text
V + 0/4 Select Until Start/End of the Line
V + B/W Select Until Start/End of the Word
V + Open/Close Bracket Select Until Start/End of the Paragraph
  • You can use either S or D as the trigger key
  • Please change your simultaneous_threshold_milliseconds setting in Karabiner-Elements → Complex Modifications → Parameters; A value between 150 and 500 is recommended for this mode
  • If you want to use another key as the trigger key, please download the source code and modify this file
