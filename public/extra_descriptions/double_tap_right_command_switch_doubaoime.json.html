This rule is designed for using DoubaoIME as a temporary voice input mode while keeping another input source for regular typing. right_command remains available as a normal modifier, and only a double tap triggers the workflow.

  • First double tap: switch to Doubao Pinyin, wait briefly, then tap right_option once to start voice input.
  • Second double tap: tap right_option once to stop voice input, wait for recognition to finish, then switch back to Rime.

By default, the second double tap switches back to Rime. To return to another input source, replace the Rime input_source_id with your preferred input source ID before importing the rule.
