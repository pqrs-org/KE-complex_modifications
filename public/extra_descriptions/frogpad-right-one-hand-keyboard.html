Right hand remap for the now-defunct Frogpad one handed keyboard.

The "J" key is remapped to the letter "H" and if you have already memorized the frogpad layout, the rest should be straightforward from there!

Special note that the < and > keys that become lost in this layout are now remapped to the left-shift+Z and left-shift+X keys respectively.

Full Keymap (Right Hand)

  • For "Primary" keys, simply type as normal with right hand.
  • To activate "Secondary" keys, hold down (spacebar) while typing.
