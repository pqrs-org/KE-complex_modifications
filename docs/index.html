Vi Mode: S + H/J/K/L = Arrow Keys; S + F + H/J/K/L = Home / Page Down / Page Up / End

  • Change S to Left_control + Fn if pressed with other keys (Post S when pressed alone)
  • Change Left_control + Fn + H/J/K/L to Arrow Keys
  • Change Left_control + Fn + F to Fn
  • Change Fn + H/J/K/L to Home / Page Down / Page Up / End
  • Change Left_control + Fn + Letter (except for F, H, J, K, L, S) to S + Letter
  • Change Shift + Left_control + Fn to Shift + S
