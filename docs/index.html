Vi Mode (rev 3)

  • Vi Mode #1 [S + H/J/K/L → Arrow Keys]
  • Vi Mode #1d [D + H/J/K/L → Arrow Keys]
  • Vi Mode #2 [S/D + F → Fn]
  • Vi Mode #3 [S/D + F + H/J/K/L → Home / Page Down / Page Up / End]
  • Vi Mode #4 [S/D + 0/4 → To the beginning/end of the line]
— #1 and #1d conflict with each other; you can use #1d instead of #1 to reduce trigger errors.
— #2 requires #1 or #1d; #3 requires #2; #4 requires #1 or #1d.
